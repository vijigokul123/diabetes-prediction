# 🩺 Diabetes Prediction using ML & Deep Learning

## 📌 Problem Statement
The objective of this project is to predict whether a patient is diabetic or not based on medical attributes such as glucose level, BMI, age, etc.

---

## 📊 Dataset
- PIMA Indians Diabetes Dataset  
- Source: https://raw.githubusercontent.com/jbrownlee/Datasets/master/pima-indians-diabetes.data.csv  
- Features: Pregnancies, Glucose, Blood Pressure, Skin Thickness, Insulin, BMI, Diabetes Pedigree Function, Age  

---

## ⚙️ Models Used

### 🔹 Machine Learning Models
- Logistic Regression  
- Decision Tree  
- Random Forest  
- SVM  
- KNN  

### 🔹 Deep Learning Model
- Artificial Neural Network (ANN)

---

## 🔄 Project Pipeline
Data Collection → Data Cleaning → Feature Scaling → ML Models → ANN Model → Comparison → Prediction

---

## 📈 Performance Evaluation
- Accuracy  
- Precision  
- Recall  
- F1 Score  
- Confusion Matrix

## 🚀 Results
